h1. Data files for jQuery.autocomplete()

Language, script and country lists for use with jQuery autocomplete. Current datasets are available:

- iso-639-3 := ISO 639-3 three letter language tags  from ISO 639-3 RA data
- lsu-languages := tags for languages from the IANA registry
- lsu-regions := tags for regions and countries from the IANA registry
- iso15924 := ISO 15924 script tags

Each file will consist of label and value pairs. The label will be the name if the langauge, script or country or region in English, while the value will be the appropriate tag.
